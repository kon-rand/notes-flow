# Models Module Documentation

## Overview

Pydantic models for data validation and serialization in the Notes Flow bot.

## Models

### InboxMessage

Represents a message in the user's inbox.

```python
class InboxMessage(BaseModel):
    id: str                    # Unique message ID (e.g., "msg_001")
    timestamp: datetime        # Message timestamp
    from_user: int             # User who sent message to bot
    sender_id: int             # Original author (for forwards)
    sender_name: Optional[str] # Original author name
    content: str               # Message text content
    chat_id: int               # Chat where message was sent
```

**Usage:**
```python
from bot.db.models import InboxMessage
from datetime import datetime

msg = InboxMessage(
    id="msg_001",
    timestamp=datetime.now(),
    from_user=123456,
    sender_id=123456,
    sender_name=None,
    content="Hello world",
    chat_id=-100123456
)
```

### Task

Represents a task created from message analysis.

```python
class Task(BaseModel):
    id: str                    # Unique task ID (e.g., "task_001")
    title: str                 # Task title
    tags: List[str]            # Task tags (max 3 recommended)
    status: str = "pending"    # "pending" or "completed"
    created_at: datetime       # Creation timestamp
    source_message_ids: List[str] # References to inbox messages
    content: str               # Full task description
```

**Usage:**
```python
from bot.db.models import Task

task = Task(
    id="task_001",
    title="Prepare report",
    tags=["work", "urgent"],
    status="pending",
    created_at=datetime.now(),
    source_message_ids=["msg_001", "msg_002"],
    content="Prepare quarterly report with all metrics"
)
```

### Note

Represents a note created from message analysis.

```python
class Note(BaseModel):
    id: str                    # Unique note ID (e.g., "note_001")
    title: str                 # Note title
    tags: List[str]            # Note tags for categorization
    created_at: datetime       # Creation timestamp
    source_message_ids: List[str] # References to inbox messages
    content: str               # Note content
```

**Usage:**
```python
from bot.db.models import Note

note = Note(
    id="note_001",
    title="Project ideas",
    tags=["ideas", "development"],
    created_at=datetime.now(),
    source_message_ids=["msg_003"],
    content="Use async/await for better performance"
)
```

## Field Descriptions

### IDs

- Format: `{type}_{number}` (e.g., "msg_001", "task_042")
- Generated by `FileManager._generate_id()`
- Unique within each file

### Tags

- List of string tags
- No strict validation (application-level)
- Recommended: 1-3 tags per item

### Status

**Task status values:**
- `"pending"` - Task not yet completed
- `"completed"` - Task finished

### Source Message IDs

- References to original inbox messages
- Format: List of message IDs (e.g., `["msg_001", "msg_002"]`)
- Used for traceability

## Validation

All models use Pydantic v2 for:
- Type validation
- Default values
- Optional fields
- Serialization/deserialization
