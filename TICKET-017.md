# TICKET-017: Замена команды /complete на /completed

## Описание
Заменить команду `/complete` на `/completed` для более корректного грамматического оформления в английском языке. "Completed" (выполненный) - более естественное название для команды просмотра архива выполненных задач.

- [ ] Обновить тесты в tests/integration/test_commands_archive.py
- [ ] Обновить документацию в PLAN.md, TICKET-016.md

## Реализация

### Изменения в handlers/commands.py

1. **Заменить декоратор команды** (строка 282):
   ```python
   @router.message(Command("complete"))
   ```
   на:
   ```python
   @router.message(Command("completed"))
   ```

2. **Обновить документацию в help-сообщениях**:
   - Строка 46: `/complete - просмотр архивов` → `/completed - просмотр архивов`
   - Строка 73: `/complete - просмотр архивов задач` → `/completed - просмотр архивов задач`

3. **Обновить сообщения об ошибках** (строка 310):
   ```python
   await message.answer("❌ Неверный формат даты. Используйте: /complete YYYY_MM_DD")
   ```
   на:
   ```python
   await message.answer("❌ Неверный формат даты. Используйте: /completed YYYY_MM_DD")
   ```
