FUNCTIONS IMPLEMENTED
---------------------
1. format_file_size(size_bytes: int) -> str
   - Converts bytes to human-readable format
   - Examples: 1024 -> "1.0 KB", 1048576 -> "1.0 MB"

2. create_zip_from_directory(source_dir: Path, zip_output: io.BytesIO) -> int
   - Creates ZIP archive from directory contents
   - Returns number of files added to archive
   - Uses standard zipfile module only

3. extract_zip_to_directory(zip_path: Path, target_dir: Path) -> int
   - Extracts ZIP file to target directory
   - Returns number of files extracted
   - Raises zipfile.BadZipFile if invalid

4. resolve_safe_path(base_dir: Path, relative_path: str) -> Path
   - Resolves safe path within base directory
   - Prevents path traversal attacks
   - Raises ValueError if path is outside base

5. list_zip_contents(zip_path: Path) -> List[str]
   - Lists all files in a ZIP archive
   - Returns list of file paths within ZIP
