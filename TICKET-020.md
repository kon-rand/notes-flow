# TICKET-020: Изменить формат вывода команды /completed

## Описание

Изменить формат вывода команды `/completed` для удобной навигации по архивам.

**Текущее поведение:**
```
📁 Архивы задач:

📅 2026-03-24 (1 задач)
```

**Ожидаемое поведение:**
```
📁 Архивы задач:

📅 /2026_03_24 (1 задач)
```

При нажатии на `/2026_03_24` пользователь переходит к просмотру архивных задач за эту дату.

## Компоненты

- [x] Изменить формат вывода даты в `/completed` — добавить префикс `/`
- [x] Добавить новый обработчик команды `/YYYY_MM_DD` для просмотра задач за дату
